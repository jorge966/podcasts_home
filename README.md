# Podcasts Home

## API Endpoints

### Users

`/create_user` - Creates a user [POST]

`/get_user/<user_id>` - Gets a user by `id` [GET]

`/delete_user/<user_id>` - Deletes a user by `id` [DELETE]

`/update_user` - Updates a user by `id` [POST]
